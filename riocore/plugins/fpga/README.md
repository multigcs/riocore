## Options:
*user-options*
### name:
name of this plugin instance

 * type: str
 * default: 

### node_type:
board type

 * type: select
 * default: Numato-Spartan6

### simulation:
simulation mode

 * type: bool
 * default: False

### speed:
FPGA clock speed

 * type: int
 * min: 1000000
 * max: 500000000
 * default: 50000000
 * unit: Hz

### flashcmd:
overwrite flash command for this instance

 * type: str
 * default:
