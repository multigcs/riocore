# arty_mii

<img align="right" width="320" src="image.png">


| :warning: EXPERIMENTAL |
|:-----------------------|

**udp interface for host comunication - Arty7-35t only**

Keywords: network ethernet interface udp

## Limitations
* boards: Arty-a7-35t
* toolchains: vivado

## Pins:
*FPGA-pins*
### phy_rx_clk:

 * direction: input
 * default: F15

### phy_rxd0:

 * direction: input
 * default: D18

### phy_rxd1:

 * direction: input
 * default: E17

### phy_rxd2:

 * direction: input
 * default: E18

### phy_rxd3:

 * direction: input
 * default: G17

### phy_rx_dv:

 * direction: input
 * default: G16

### phy_rx_er:

 * direction: input
 * default: C17

### phy_tx_clk:

 * direction: input
 * default: H16

### phy_txd0:

 * direction: output
 * default: H14

### phy_txd1:

 * direction: output
 * default: J14

### phy_txd2:

 * direction: output
 * default: J13

### phy_txd3:

 * direction: output
 * default: H17

### phy_tx_en:

 * direction: output
 * default: H15

### phy_col:

 * direction: input
 * default: D17

### phy_crs:

 * direction: input
 * default: G14

### phy_ref_clk:

 * direction: output
 * default: G18

### phy_reset_n:

 * direction: output
 * default: C16


## Options:
*user-options*
### mac:
MAC-Address

 * type: str
 * default: AA:AF:FA:CC:E3:1C

### ip:
IP-Address

 * type: str
 * default: 192.168.10.194

### port:
UDP-Port

 * type: int
 * default: 2390

### name:
name of this plugin instance

 * type: str
 * default:
